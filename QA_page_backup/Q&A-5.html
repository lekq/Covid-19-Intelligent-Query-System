  • How to Save Interested Articles in A Repository?

    • Create a repository on the left panel with "New Repository".
    • Click "Interested" button to mark the article(s).
    • Click the "Repository Management" to save all interested articles in the corresponding repository.

  • How to Manage Repository?

    • Clear all articles in a repository with icon (∅) in the left panel.
    • Rename a repository with icon () in the 'Repository Management' menu in the right panel.
    • Delete a repository with icon () in the 'Repository Management' menu in the right panel.

  • How to Download Articles?

    • Full Text: you can click 'Full Text' button around the title to the publisher's website then download the article if you have premission.
    • Citation Download: please use the gray 'Citation' button around the title to download .bilb file.
