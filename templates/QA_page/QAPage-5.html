  • How to Save Interested Articles in A Repository?

    • Choose one query mode and decide your input type. 
    • View article preview to decide if it is pertinent to your line to research.
    • Broaden or narrow down the searching according to the number of article return in the green strip on the top. 
    • Choose your filter preferences on the left panel. 
    • Click "Interested" to mark the articles you are going to save. 
    • Create a repository on the left panel with "New Repository". 
    • Click the "Repository Management" to save the interested articles in the corresponding repository. 

  • How to Edit Repository?

    • If you want to manage articles in the repository, please go to the panel on the left; if you want to manage the repository itself, please go to the “repository management” button on the upper right corner. 
    • View articles in a saved repository in the left panel. 
    • Clear articles in a repository in the left panel (∅).
    • Rename the repository name in the right panel ().
    • Delete the repository in the right panel ().

  • How to Download Articles?

    • Export PDFs: you can only export PDFs of all the articles in all the repositories you created, to do so please find the "Export all records" on the left panel. 
    • Download citations: please use the gray button in the title.
